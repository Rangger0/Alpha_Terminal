# Alpha Terminal

Portfolio bergaya premium untuk menampilkan:

- build Web3
- tracking airdrop
- catatan keuangan
- roadmap dan README-style project notes

## Direction

Brand ini diarahkan menjadi "portfolio operating system", bukan sekadar landing page biasa. Karena itu desainnya memakai:

- identitas baru `Alpha Terminal`
- palet obsidian, cyan, dan amber
- card system yang lebih mewah dan editorial
- project cards dengan preview, README, dan roadmap

## Sections

- `Home`: positioning, workflow, featured nodes, media shortcuts
- `Projects`: project vault dengan filter, preview panel, README notes, dan roadmap
- `About`: operator profile, principles, stack matrix, contact console
- `Media`: external channel hub yang lebih rapi dan terstruktur

## Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React

## Local Development

```bash
npm install
npm run dev
```

Build production:

```bash
npm run build
```

Lint:

```bash
npm run lint
```
